# Projeto Gtech

Projeto Next.js criado com React, TypeScript, App Router e Tailwind CSS.

## Comandos

Instale as dependencias:

```bash
npm install
```

Rode em desenvolvimento:

```bash
npm run dev
```

Gere o build de producao:

```bash
npm run build
```

Inicie o servidor de producao:

```bash
npm start
```
